# dosbox-vim

A old-school DOSBOX-like theme for vim that might induce nostalgia.

# Installation

Using [vim-plug](https://github.com/junegunn/vim-plug):

Add

`Plug 'gsmrs/dosbox-vim-colorscheme'`

to your `.vimrc`'s plugin section.
